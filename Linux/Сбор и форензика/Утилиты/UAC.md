Сбор артефактов с помощью утилиты UAC

Unix-like Artifacts Collector (UAC) – это утилита для сбора первичных артефактов с жесткого диска.

Выполните следующие шаги:

1. Загрузите утилиту по ссылке https://github.com/tclahr/uac/releases/tag/v3.0.0.

2. Запустите утилиту от имени администратора. Команда для запуска в консоли с правами root:
`./uac -p ir_triage -f zip /tmp/uac`

3. Дождитесь выполнения команды. Появится архив по пути /tmp/uac/uac-*.zip.
